Gestion de l'agenda

Configuration initiale

Pour enregistrer les modifications sur le site, entrez votre jeton GitHub une seule fois. Il sera mémorisé sur cet appareil.

  1. Allez sur github.com → Settings → Tokens
  2. Cliquez Generate new token
  3. Nom : Saint-Roch Agenda
  4. Repository : saint-roch-evenements
  5. Permissions : Contents → Read and Write
  6. Copiez et collez le jeton ci-dessous

Gestion des disponibilités

Disponible — cliquer pour réserver
Réservé — cliquer pour voir/modifier
En option — cliquer pour libérer
📍 Cliquez maintenant sur la date de fin — Échap pour annuler

Cliquez sur la 1ère date pour commencer, puis sur la date de fin pour valider. Cliquez sur une date réservée pour voir la fiche.
