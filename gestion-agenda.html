Gestion de l'agenda

Configuration initiale

Pour que les modifications soient enregistrées sur le site, entrez votre jeton GitHub une seule fois. Il sera mémorisé sur cet appareil.

  1. Allez sur github.com → Settings → Tokens
  2. Cliquez Generate new token
  3. Nom : Saint-Roch Agenda
  4. Repository : saint-roch-evenements
  5. Permissions : Contents → Read and Write
  6. Copiez et collez le jeton ci-dessous
